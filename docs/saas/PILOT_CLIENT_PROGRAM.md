# Pilot Client Program

**Duration:** 3 months
**Target:** 3 manufacturing clients in Karachi
**Discount:** 50% off Starter tier (PKR 37,500/month)

---

## Selection Criteria

| Criteria | Required |
|---|---|
| Industry | Glass, steel, aluminum, marble manufacturing |
| Size | 30-100 employees |
| Current system | Excel / WhatsApp / outdated ERP |
| Champion | Owner or Finance Director |
| Location | Karachi (in-person support) |
| Commitment | 3-month contract, weekly feedback calls |

## Pilot Offer

- PKR 37,500/month (50% of PKR 75K Starter)
- Free onboarding (normally PKR 50K)
- Free data migration (up to PKR 100K value)
- Weekly 30-min check-in call with Hassan
- Direct WhatsApp support line (24hr response)
- Feature requests prioritized in development

## Success Metrics

| Metric | Target | Measurement |
|---|---|---|
| Onboarding time | < 4 hours | Clock from signup to first event |
| EventOS accuracy (month 1) | 70%+ | Correct classifications / total |
| EventOS accuracy (month 3) | 85%+ | With learning loop improvements |
| User adoption | 60%+ staff using daily | Active users / total users |
| Client retention | 2/3 renew at full price | Contract renewal rate |
| Testimonial | 1 public reference | Video or written testimonial |
| Critical bugs | 0 causing downtime | Incident tracking |
| ROI proof | PKR 100K+ savings in 3 months | Before/after comparison |

## Pilot Timeline

| Week | Activity |
|---|---|
| 1 | Company setup, COA generation, master data import |
| 2 | EventOS training (10 common events), staff orientation |
| 3-4 | Supervised operation (Hassan monitors all agent decisions) |
| 5-8 | Semi-autonomous (agents recommend, owner approves) |
| 9-12 | Full operation + feedback collection |
| 12 | Renewal decision, testimonial recording |

## Exit Criteria (Fail)

- Client requests cancellation before month 3
- >5 critical bugs in first month
- EventOS accuracy stays <50% after 20 events
- Owner stops engaging with feedback calls
